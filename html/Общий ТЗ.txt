Дизайн сайта для работы над книгой

Меню глав и разделов
У каждой главы раздела статистика: кол-во слов/букв, дата правки + ссылка на саммари данного раздела/главы


Соавторы - это будут ИИ-агенты с аватарками и промптами
Страница редактирования главы/раздела 
Страница текста раздела. Сам текст + блоки комментариев от ИИ агентов

Технически сайт должен состоять из .html файлов в корневой папке + вспомогательные файлы css. Один общий сss для общих для всех страниц элементов + для каждой страницы свой отдельный css в котором описаны стили для этой страницы. Все фильтр - это ссылки на страницы с добавлением get параметров (например ?city=london)

Общая цель сайта: Для удобства написания книги + возможность писать вместе с ИИ агентами., делать саммари по главам, отдавать их ИИ чтобы они понимали контекст всей книги когда работают с главной. Желательно иметь возможность создавать обсуждение нескольких ИИ-агентов над лавой (мультиагентный консенсус экспертов), добавлять планы, замечания по книге, добавлять и отслеживать идеи и замечания.

Создай столько html и css файлов сколько нужно для создания сервиса удобного планирования и написания книги
